# Ember & Ash

Ember & Ash is a demo restaurant web application built with Next.js. It showcases a polished customer experience for browsing menu items, placing simulated orders, booking reservations, and exploring a restaurant admin dashboard in a warm fire-themed interface.

## Features

- Browse a curated Nigerian-inspired menu with categories, featured items, and menu cards.
- Add items to cart, update quantities, and review pricing before checkout.
- Simulate checkout with delivery, pickup, and dine-in order flows.
- Use card, bank transfer, or cash payment simulation during checkout.
- Book reservations with a clean reservation form and WhatsApp handoff.
- View the restaurant contact details, opening hours, and location information.
- Sign in with demo/local auth to access the admin dashboard.
- Manage orders, reservations, and menu views from the admin area.

## Demo Notes

This is a portfolio and demo project, not a production ordering platform.

- Authentication is local/demo based.
- Payment methods are simulated for presentation purposes.
- Order placement is intended to demonstrate the flow and UI, not real payment processing.
- Reservation and WhatsApp actions are designed to show how a real restaurant app could work.
